# Bank Transaction Monitoring System (SQLite)

This repository contains a **Bank Transaction Monitoring System** implemented using **SQLite**.  
The project is designed to store, track, and analyze banking transactions efficiently, making it suitable for learning database design and transaction management concepts.

## Features
- Customer account management
- Transaction recording (deposits, withdrawals, transfers)
- Transaction history tracking
- Lightweight and fast SQLite database
- Simple and scalable schema design

## Technologies Used
- SQLite
- SQL

## Use Cases
- Academic and practice projects
- Learning financial database systems
- Portfolio and interview-ready SQL project
- Transaction monitoring and reporting practice

## Project Structure
- Database schema for accounts and transactions
- Sample data for testing and practice
- Query examples for monitoring transactions
